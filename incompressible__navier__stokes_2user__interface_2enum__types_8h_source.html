38/*
39 * ProblemType refers to the underlying physics of the flow problem and describes
40 * whether the considered flow problem is expected to be a steady or unsteady solution
41 * of the incompressible Navier-Stokes equations. This essentially depends on the
42 * Reynolds number but, for example, also on the boundary conditions (in case of
43 * time dependent boundary conditions, the problem type is always unsteady).
44 */

46{
47 Undefined,

49 Unsteady
50};

108 * SolverType refers to the numerical solution of the incompressible Navier-Stokes
109 * equations and describes whether a steady or an unsteady solver is used.

166/*
167 * Pseudo-timestepping for steady-state problems:
168 * Define convergence criterion that is used to terminate simulation
169 *
170 * option ResidualSteadyNavierStokes:
171 * - evaluate residual of steady, coupled incompressible Navier-Stokes equations
172 * and terminate simulation if norm of residual fulfills tolerances
173 * - can be used for the coupled solution approach
174 * - can be used for the pressure-correction scheme in case the incremental
175 * formulation is used (for the nonincremental formulation the steady-state
176 * solution cannot fulfill the residual of the steady Navier-Stokes equations
177 * in general due to the splitting error)
178 * - cannot be used for the dual splitting scheme (due to the splitting error
179 * the residual of the steady Navier-Stokes equations is not fulfilled)
180 *
181 * option SolutionIncrement:
182 * - calculate solution increment from one time step to the next and terminate
183 * simulation if solution doesn't change any more (defined by tolerances)
184 */
185enum class ConvergenceCriterionSteadyProblem
186{
187 Undefined,
188 ResidualSteadyNavierStokes,
189 SolutionIncrement
190};

218/*
219 * Type of imposition of Dirichlet BC's:
220 *
221 * direct: u⁺ = g
222 * mirror: u⁺ = -u⁻ + 2g
223 *
224 * We normally use the option Mirror as default setup.
225 * A direct imposition might be advantageous with respect to the CFL condition
226 * possibly allowing to use larger time step sizes (approximately 20 percent)
227 * depending on other parameters of the spatial discretization (divergence versus
228 * convective formulation of convective term, upwind factor, and use of divergence
229 * and continuity penalty terms).
230 */
231enum class TypeDirichletBCs
232{
233 Direct,
234 Mirror
235};
