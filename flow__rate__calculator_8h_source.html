52 /*
53  * This class calculates a vector of flow rates where the different entries of the vector correspond
54  * to different boundary IDs, i.e., one outflow boundary may only consist of faces with the same
55  * boundary ID. This class is intended to be used in cases where the number of outflow boundaries is
56  * very large with a typical use case being the human lung (where the number of outflow boundaries
57  * is 2^{N-1} with N being the number of airway generations). The reason behind is that calculating
58  * the flow rate requires global communication since different processors may share the same outflow
59  * boundary and the implementation in this class only requires one communication for all outflow
60  * boundaries.
61  *
62  * Note: For a small number of outflow boundaries (for example 1-10), the more modular class
63  * MeanVelocityCalculator should be used rather than this specialized implementation.
64  */
